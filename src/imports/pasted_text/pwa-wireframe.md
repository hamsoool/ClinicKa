Create a detailed wireframe for a Progressive Web App (PWA) called:

"Gordon College Health Services Unit – Digital Medical Record System"

Use Gordon College dark green as the primary color.
Primary color: #1a5f3f
Secondary green: #2d9963
Light background: #f0faf4

Design should look modern, clean, and suitable for a university medical system.

The system has TWO ROLES:

1. Student
2. Clinic Staff

Use a dashboard layout with side navigation for staff and step-based forms for students.
